# Discord.py AI 機器人

這是一個使用 Discord.py 建立的 AI 機器人。

## 功能

- 自動回應訊息
- 管理伺服器
- 提供遊戲資訊
- ...（請根據您的機器人功能進行修改）

## 安裝

1. 克隆此存儲庫
2. 安裝依賴項：`pip install -r requirements.txt`
3. 設定您的機器人 token 在 `config.py`
4. 運行 `bot.py`

## 使用

請在 Discord 中輸入 `!help` 以獲取有關如何使用機器人的更多資訊。
